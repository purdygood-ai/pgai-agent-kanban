# Dashboard Layout Tests

Inventory of tests carrying the `@pytest.mark.dashboard_layout` marker.

These tests assert on rendered visual output: column counts, pane widths,
character counts, and arithmetic that governs how the dashboard is laid out.
The marker is informational — tests run by default and are not skipped.

To run only these tests:

```
cd team/
pytest --strict-markers -m dashboard_layout
```
